This page describes a demo for a reproductive L\+S\+PG R\+OM applied to a 1D advection-\/diffusion problem using P\+OD modes as basis. By the end, it should be clear how to setup the problem and the various steps involved.~\newline
 To jump directly at the full demo script, click \href{https://github.com/Pressio/pressio4py/blob/master/demos/unsteady_default_lspg_advdiff1d_pod/main.py}{\texttt{ here.}}

\begin{DoxyParagraph}{Disclaimer}
This demo purposefully focuses on a simple test since the main goal is to demonstrate the steps and the code. More complex cases will be added later.

\hypertarget{md_pages_demos_demo2_autotoc_md10}{}\doxysection{Overview}\label{md_pages_demos_demo2_autotoc_md10}
Here, we cover the three typical steps needed for a R\+OM\+:
\begin{DoxyEnumerate}
\item generating of snapshots using the full-\/order model (F\+OM)
\item computing the basis\+: here we demonstrate the use of P\+OD modes
\item executing the R\+OM\+: here we leverage the L\+S\+PG R\+OM to demonstrate a {\itshape reproductive} test, i.\+e., we run the R\+OM using the same physical coefficients, b.\+c., etc. A predictive run is demonstrated in a different tutorial.
\end{DoxyEnumerate}\hypertarget{md_pages_demos_demo2_autotoc_md11}{}\doxysection{F\+O\+M Equations}\label{md_pages_demos_demo2_autotoc_md11}
The governing equations for this problem are\+:

\[ \frac{\partial u}{\partial t} = \frac{\partial}{\partial x} (k(u,x) \frac{\partial u}{\partial x} ) - a*\frac{\partial u}{\partial x} \] where $k(u,x)=x^4$, the field is $u(x;t)$, the advection velocity is fixed at $a=2$, the spatial coordinate is $x$ and the domain is $(0,1)$. We use homogeneous BC. Note that a class approximating the F\+OM operators via finite-\/differences is implemented \href{https://github.com/Pressio/pressio4py/blob/master/apps/adv_diff1d.py}{\texttt{ here}}.\hypertarget{md_pages_demos_demo2_autotoc_md12}{}\doxysection{Imports}\label{md_pages_demos_demo2_autotoc_md12}
